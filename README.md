# luhn-algorithm-project

This was a project created for my discrete structures class in fall 2024. It generates random credit card numbers and uses the Luhn algorithm to verify if they are valid. The project will generate a credit card number, flip a random digit on the number to a random value 1-9 and then check to see if the card is still valid using the luhn algorithm.

##installation

  1. compile the cpp file using a C++ compiler
  2. run the created exe file
